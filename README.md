# Anthropic 2026 Launches

Every product, launch, and update Anthropic has shipped in 2026 — from Claude Opus 4.6 to Mars.

Live timeline dashboard with dark mode, search, filters, and click-through details.

## Stack

Vite + React 18 + TypeScript + Tailwind + Recharts + framer-motion.

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5190](http://localhost:5190).

## Build

```bash
npm run build
```
